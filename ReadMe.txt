- The config file needs to retain the same structure. It reads in data in the order provided in the file.

- The filepath in the config file needs to be the path to the metadata where ever that may be

- The processor scheduling can be changed but it needs to be in the following format:

	FIFO - if you want first in first out
	SJF  - if you want shortest job first
	Round Robin - if you want round robin

- The log variable needs to be used to print to console, file or both
